# LiFi Bridge - Cross-Chain Token Transfer App

## Overview

LiFi Bridge is a powerful cross-chain bridge application that enables seamless token transfers between different blockchain networks. Built with Next.js and integrated with the LiFi protocol, this app provides users with a secure, efficient, and user-friendly way to bridge their digital assets across multiple chains.

## Key Features

- **Cross-Chain Bridging**: Transfer tokens between different blockchain networks seamlessly
- **Multi-Chain Support**: Access to a wide range of blockchain networks
- **LiFi Integration**: Powered by the LiFi protocol for optimal routing and execution
- **Dual Platform Support**: Available as both a Farcaster Mini App and Progressive Web App (PWA)
- **Mobile-First Design**: Optimized for mobile devices with responsive navigation
- **Safe Area Management**: Perfect fit for mobile screens with proper safe area handling

## Supported Chains

### Primary Swap Networks

**Base** - Ethereum L2 scaling solution offering fast, low-cost transactions with full EVM compatibility
- Low gas fees compared to Ethereum mainnet
- Fast transaction finality
- Secure and decentralized

**Celo** - Mobile-first blockchain designed for financial inclusion
- Carbon-negative consensus mechanism
- Built-in stablecoin support
- Mobile-optimized infrastructure

### Additional Networks
The app supports many other blockchain networks through the LiFi protocol, providing users with comprehensive cross-chain functionality.

## Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS with custom theming
- **Blockchain Integration**: LiFi SDK and Widget
- **Deployment**: Optimized for both web and mobile platforms
- **State Management**: React hooks with proper hydration handling

## Getting Started

### Prerequisites
- Node.js 18+ or Bun
- Modern web browser or mobile device
- Web3 wallet (MetaMask, WalletConnect, etc.)

### Installation
```bash
# Clone the repository
git clone [repository-url]

# Install dependencies
npm install
# or
bun install

# Run development server
npm run dev
# or
bun dev
```

### Building for Production
```bash
# Build the application
npm run build

# Start production server
npm start
```
